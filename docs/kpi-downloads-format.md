# Format of `data/kpi_downloads.json`

Generated by `scripts/GenerateDownloadKpiCommand.php` via `make stats-download-kpi`.

## Regeneration

```bash
make stats-download-kpi pretty=1             # all journals, pretty-printed JSON
make stats-download-kpi rvcode=epiga         # one journal only
make stats-download-kpi output=/srv/kpi.json # custom output path
make stats-download-kpi dry-run=1            # console summary, no file written
```

## TypeScript Schema

### Root

```ts
type KpiFile = {
  generated_at:   string;                   // ISO 8601, e.g. "2026-04-06T14:00:00+00:00"
  total_papers:   number;                   // total number of published papers with a DOI
  total_journals: number;                   // number of journals in the file
  journals:       Record<string, Journal>;  // key = rvcode (e.g. "epiga")
};
```

### Journal

```ts
type Journal = {
  rvid:         number;   // numeric journal ID
  name:         string;   // human-readable journal name
  papers_count: number;   // number of papers in this journal
  papers:       Paper[];
};
```

### Paper

```ts
type Paper = {
  doi:              string;        // canonical DOI, e.g. "10.46298/epiga.2024.123"
  paperid:          number;        // logical identifier (all versions combined)
  publication_date: string | null; // "YYYY-MM-DD" or null
  downloads:        Metric;        // PDF downloads (CONSULT = 'file')
  page_views:       Metric;        // abstract page views (CONSULT = 'notice')
  geo:              GeoMap;        // geographic breakdown of visits
};
```

### Metric

```ts
type Metric = {
  total:   number;                  // cumulative total over the entire period
  by_year: Record<string, number>;  // key = "YYYY", value = number of human visits
};
```

### GeoMap

```ts
// Key = ISO 3166-1 alpha-2 country code (e.g. "FR", "US") or "" when country is unknown.
// The object is {} (not []) when there are no geolocated visits.
type GeoMap = Record<string, GeoEntry>;

type GeoEntry = {
  continent:  string; // GeoIP continent code (e.g. "EU", "NA") or "" when unknown
  downloads:  number; // number of PDF downloads from this country
  page_views: number; // number of abstract page views from this country
};
```

## Business Rules

| Rule | Detail |
|------|--------|
| **Bots excluded** | `ROBOT = 0` in PAPER_STAT — filtered by `stats:process` |
| **Multi-version** | Stats from all DOCIDs of the same PAPERID are summed; `paperid` is the stable identifier |
| **Time granularity** | Annual (`by_year` key = `"YYYY"`) to keep file size manageable |
| **Geography** | By country (ISO alpha-2) with continent; unknown country = key `""` |
| **Journal sort** | Alphabetical by rvcode |
| **Country sort** | Alphabetical within `geo` (empty key `""` last) |
| **No stats** | `total: 0`, `by_year: {}`, `geo: {}` — always present |

## Useful Next.js Patterns

```ts
// Iterate over all papers in a journal
kpi.journals["epiga"].papers.forEach(paper => { ... });

// Top 5 countries by downloads for a paper
const top5 = Object.entries(paper.geo)
  .filter(([country]) => country !== "")
  .sort(([, a], [, b]) => b.downloads - a.downloads)
  .slice(0, 5);

// Annual trend (for a chart)
const years = Object.keys(paper.downloads.by_year).sort();
const values = years.map(y => paper.downloads.by_year[y]);

// Total worldwide downloads for a journal
const total = journal.papers.reduce((s, p) => s + p.downloads.total, 0);
```
